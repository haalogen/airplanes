Computational routines

  • Image shape
    • MCreateMosaicShape — Create a mosaic image shape.
  • Projection onto image shape
    • MCalculateNorm — Calculate norm of the given image.
    • MCalculateProjectionNorm — Calculate norm of the projection of the given image.
    • MCreateProjection — Create the projection onto the given image shape.
    • MPrepareNorm — Prepare to calculate norm of an image.
    • MPrepareProjectionNorm — Prepare to calculate norm of an image projection.
    • MProject — Project an image onto the image shape using the specified projection.
