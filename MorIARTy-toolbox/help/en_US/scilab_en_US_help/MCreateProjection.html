MCreateProjection

Create the projection onto the given image shape.

Calling Sequence

projection = MCreateProjection(shape)

Parameters

shape:

Structure representing the image shape.

projection:

Structure representing the created projection.

Description

MCreateProjection() returns a structure representing the projection onto the given image shape. The argument shape must be a structure representing the image shape. Such a stucture is typically created using one of the MorIARTy shape creation functions like MCreateMosaicShape().

Examples

// Read an image
imgfile = fullpath(MorIARTyPath() + "/images/cube/cube-1.png");
cube = im2double(imread(imgfile));
cube = cube(:, :, 1);

// Create the image shape
shape = MCreateMosaicShape(cube, "grayscale");

// Create the projection onto the shape
projection = MCreateProjection(shape);
