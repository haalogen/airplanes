MQuantize

Quantize data using specified quantization levels.

Calling Sequence

qdata = MQuantize(data, levels)

Parameters

data:

Input data.

levels:

Quantization levels.

qdata:

Output data.

Description

MQuantize() quantizes the given data using specified quantization levels contained in the N-element vector levels. Output data is of the same size as input data and contains N + 1 discrete integer values in the range 1 to N + 1 which are determined by the following criteria:

  • If data(k) ≤ levels(1), then qdata(k) = 1.
  • If levels(m-1) < A(k) ≤ levels(m), then qdata(k) = m.
  • If data(k) > levels(N), then qdata(k) = N + 1.

Examples

data = 1 : 10;
levels = [3 7];
qdata = MQuantize(data, levels)
